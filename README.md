# Overview
In this project, the popular titanic dataset is used to build a KMeans model to predict if a passenger will survive on the titanic.

# Dependencies
* Numpy
* Pandas
* Matplotlib
* Scikit-Learn

# Usage
1. `cd 'Predicting Survival of Titanic Passengers' `
1. `python 'KMeans model.py'`
